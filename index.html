Hexo搭建静态博客

Hexo 是一款基于 Node.js 的快速的、简单的博客框架，能够创建一个项目进行编辑后生成一套静态网页，比较适合个人博客搭建。因为 Hexo 生成的网页不依赖数据库和任何 Web 工具，所以可以把它放在 Github上，然后配置好域名以后进行访问。

1.安装前置条件

安装 Hexo 相当简单。然而在安装前，你必须确定你安装了node.js和git

5.发布到git上

  • 下载hexo-deployer-git插件

  • 修改_config.yml(blog根目录下)

  • 在github上新建仓库

  • 部署

  • 每次部署的步骤，可按以下三步来进行。

###6. 发布文章

  • 将内容写入一个md格式的文件内
  • 在文件所在文件夹下打开命令窗口
  • hexo generate/g
